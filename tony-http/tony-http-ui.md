# Tony Http UI

基于tony-http-annotation自定义注解自动生成文档，采用插件方式部署，通过jarslink实现类加载隔离，spring容器隔离，通过MDC和SiftingAppender实现日志隔离。
支持：
+ 插件式热部署
+ jar依赖分析
+ 接口文档自动生成
+ 接口测试
+ 接口Mock
+ 执行日志在线查看
+ ......
